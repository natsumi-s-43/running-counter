# Monthly Running Distance Tracker

A simple application that helps users track and manage their total running distance each month.  
This app allows runners to record daily distances and automatically calculates the monthly total.

## Demo
https://natsumi-s-43.github.io/running-counter/

## Features

- Add daily running distance
- Automatically calculate total distance for the current month
- Clear and easy-to-use interface
- Data organized by month
- Responsive design for desktop and mobile

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This project was created as a portfolio project to practice:
- JavaScript logic and data handling
- DOM manipulation and event handling
- Building a practical, real-life tracking application
- Creating a clean and user-friendly UI
